# Лабораторна робота №1. Фронтенд

## Варіант №2

**Тема:** Трекер інцидентів кібербезпеки. 
**Ключові сутності:** Заявник (Reporter), Дата/Час, Тег (Tag), Критичність (Severity), Коментар.

## Функціонал

* **Додавання інцидентів:** Форма з повною валідацією обов'язкових полів.
* **Відображення:** Динамічне виведення даних у таблицю (`<thead>` та `<tbody>`).
* **Фільтрація та сортування:** Пошук за ім'ям заявника, фільтр за рівнем критичності та сортування за датою.
* **Додатково:** Видалення записів, редагування існуючих інцидентів та збереження всіх даних у `localStorage`.

## Запуск

Проєкт розроблено як статичний фронтенд і може бути запущений локально.

1. Відкрийте файл `index.html` у вашому браузері.
2. Або скористайтеся розширенням **Live Server** у VS Code: натисніть правою кнопкою миші на `index.html` та оберіть "Open with Live Server".
